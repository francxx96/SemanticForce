SemanticForce is our main function! It allows you to perform all the operations sequentially (Pipeline), then passing through the Crawling to extract the url from the inserted page, the Scarping of the associated texts, the Named Entity Recnognition of these texts and finally the queries to Wikidata to search for the exact associations to these entities.

The Crawling is the feature that allows you to automatically analyze Web content by identifying all the hyperlinks related to the URL and then visiting all the links. The final product is an index of web page URLs.

Scraping is a function of extracting data from websites. In particular it extracts TITLE, AUTHOR and TEXT from the web page passed by URL.

EntityRecognition consists of searching for unstructured text to identify and classify the entity named in predefined categories in particular: names, organizations, locations, temporal expressions, percentages, monetary values, dates.

Wikidata allows you to make a request to the service by sending it an entity name. It returns the corresponding associations, in particular the ten best results.
