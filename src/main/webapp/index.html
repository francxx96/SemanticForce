Extract the info from article URL

Data extraction is the act or process of retrieving data out of data sources for further data processing or data storage. The import into the intermediate extracting system is thus usually followed by data transformation and possibly the addition of metadata prior to export to another stage in the data workflow. Typical unstructured data sources include web pages, emails, documents, PDFs, scanned text, mainframe reports, etc.. The functionality offered by this section allows you to extract textual content from a journalistic article on a website.

Web Crawler Extraction

Web Crawling, also called Spidering, is the process of finding Web pages and downloading them. At the same time the Web Crawler also known as Spider or Robot, is a program that downloads Web pages associated with URLs indicated, extracts the hyperlinks contained in them and continuously downloads the Web pages found by these links. Over a period of time, a significant portion of the "Surface Web" is scanned and links are extracted from it. The functionality offered by this section allows to obtain, starting from a source URL, all the hyperlinks present within it.
