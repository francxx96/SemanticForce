Named entity recognition is an activity based on the identification of entities which are classified
into predefined categories such as the names of people, organizations, places, quantities, percentages, etc.



In order to recognize entities, there are two problems to be solved.The first is to identify the names
and the second to classify them according to the type of entity to which they refer.The two main tasks to solve this are:

• SEGMENTATION : phase in which entities must be identified by names defined as tokens;
• CLASSIFICATION : the segments identified in the previous point are classified
